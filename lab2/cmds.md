# Important commands

### Number of interfaces and related information
```bash
# info
ifcongig
man ifconfig

ping <domain name>
ping <ip>

# domain to ip
host google.com

# ip to domain
host 198.283.392.239

# To ping for specifc number of times use -c
ping <domain name/ip> -c 5

# find path
traceroute

# routing table
netstat -r
route

# dns
dig manipal.edu
nslookup manipal.edu

# arp table
arp

# wireless config info
iwconfig

hostname

whois manipal.edu
```
